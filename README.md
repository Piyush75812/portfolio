# Piyush Kumar – Developer Portfolio

## Folder Structure
```
portfolio/
├── index.html          ← Main HTML file (all 10 sections)
├── style.css           ← Custom styles + CSS variables
├── script.js           ← All JS features (15 functions)
├── README.md           ← This file
└── assets/
    ├── images/         ← Place your own images here
    │   └── .gitkeep
    └── Piyush_Kumar_MERN_Stack_Developer_Resume.pdf ← Add your resume PDF here
```

## CDN Links Used
- Bootstrap 5.3.3 CSS + JS Bundle
- Bootstrap Icons 1.11.3
- Google Fonts: Space Grotesk + JetBrains Mono

## Setup Instructions
1. Unzip / open the `portfolio/` folder
2. Replace the Unsplash image URLs with your own photo
3. Add your resume PDF → `assets/resume/Piyush_Kumar_MERN_Stack_Developer_Resume.pdf`
4. Update contact details in `index.html` (phone, email, GitHub, LinkedIn)
5. Open `index.html` in any browser – no build step required!

## To deploy
- GitHub Pages: push to a repo → Settings → Pages → main branch
- Netlify / Vercel: drag and drop the folder
